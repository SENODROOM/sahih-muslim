# Data Format — muslim.json

## data/muslim.json shape

```json
{
  "metadata": {
    "id": 1, "length": 7563,
    "arabic":  { "title": "صحيح مسلم", "author": "الإمام مسلم بن الحجاج" },
    "english": { "title": "Sahih Muslim", "author": "Imam Muslim ibn al-Hajjaj" }
  },
  "chapters": [ { "id": 1, "arabic": "كتاب الإيمان", "english": "Faith" } ],
  "hadiths": [
    { "id": 1, "chapterId": 1, "arabic": "...", "english": { "narrator": "...", "text": "..." } }
  ]
}
```

## data/chapters/ folder
Generated by `node scripts/build.mjs`. Do not edit manually.
- `meta.json` — metadata + chapters list
- `{id}.json` — hadiths for each chapter
